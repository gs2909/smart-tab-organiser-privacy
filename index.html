Privacy Policy - Smart Tab Organizer

Effective date: June 13, 2026

Smart Tab Organizer is a Chrome extension for organizing browser tabs, saving tab sessions, restoring workspaces, searching tabs, and cleaning up duplicate tabs.

This public GitHub Pages site and its public repository exist only to host this privacy policy for Chrome Web Store review and user access.

Information Handled By The Extension

Smart Tab Organizer may handle the following information in order to provide its tab management features:

  • Open tab URLs, titles, domains, favicons, and browser tab metadata
  • Saved tab sessions and session history
  • User-created workspace or grouping information
  • Extension settings
  • License key and license validation status if the user purchases lifetime access
  • Optional AI provider configuration, including API keys if the user chooses to enter them

How The Information Is Used

This information is used only to provide and improve the extension's tab management features, including tab review, grouping, duplicate detection, saved sessions, session restoration, search, import/export, and optional AI-assisted organization suggestions.

Local Storage

Smart Tab Organizer is local-first. Saved sessions, settings, workspace data, and optional provider settings are stored using Chrome extension storage. The core extension does not require an account or login.

Payments And License Validation

Smart Tab Organizer may offer a free trial followed by a paid lifetime license. Payments and license key delivery are handled by Dodo Payments, not Google.

If a user enters a license key, the extension sends that user-entered license key to Dodo Payments' public license endpoints to validate and activate access. The extension stores license status locally so the user can continue using the paid version after successful validation.

The Dodo Payments host permission is used only for license validation and activation. It is not used to collect browsing history, tab contents, saved sessions, or unrelated website data.

Optional AI Features

AI features are optional and disabled unless configured by the user. If the user enables AI suggestions and chooses an AI provider, selected tab context needed for the requested suggestion may be sent to the provider configured by the user, such as OpenAI, Anthropic, Google Gemini, or OpenRouter. The core tab management features work without AI.

Users should review the privacy terms of any AI provider they choose to connect.

Sharing

The developer does not sell user data. The extension does not send tab data to the developer's own server. Data is shared with third parties only when necessary to provide a user-enabled feature, such as optional AI suggestions through the provider selected by the user, payment processing and license validation through Dodo Payments, or when required by law.

Security

License validation and optional AI provider requests are sent over HTTPS. Users are responsible for managing any API keys they choose to store in the extension.

Limited Use

The use of information received from Google APIs will adhere to the Chrome Web Store User Data Policy, including the Limited Use requirements.

Data Control

Users can remove saved sessions, change settings, export/import extension data, and uninstall the extension to remove extension-managed local data from Chrome storage according to Chrome's extension data behavior.
